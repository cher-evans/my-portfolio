Hi, I’m Cheruiyot Evans — a junior full-stack developer at the beginning of an exciting journey into the world of web development

I started by mastering the core building blocks of the web: HTML and CSS, crafting clean and responsive layouts. As I deepen my understanding of JavaScript, I'm steadily progressing toward becoming a full-stack developer.

services

web development

I build websites using modern web technologies, focusing on both function and performance. From structuring content with HTML, styling with CSS, and adding interactivity with JavaScript, I turn designs into fully working, responsive websites that are fast and accessible.

web design

I create clean, responsive, and visually appealing website designs that provide a great user experience across all devices. My goal is to design layouts that are not only beautiful but also easy to navigate and aligned with your brand’s identity.

app design

I design mobile and web application interfaces that are intuitive, user-friendly, and visually engaging. Whether it’s wireframing an idea or creating polished screens, I focus on delivering designs that make the user’s experience simple and enjoyable.

skills

html & css

I have a solid foundation in HTML5 and CSS3, using best practices to write clean, semantic, and accessible code. I’m comfortable building well-structured web pages and layouts using Flexbox and Grid, and I pay close attention to design details, spacing, and consistency. I also understand how to organize CSS for maintainability and scalability as projects grow.

javaScript

I’m currently developing my skills in JavaScript and have a good grasp of core concepts like variables, functions, arrays, conditionals, loops, and basic DOM manipulation. I’ve used JavaScript to add simple interactivity to web pages — such as toggle menus and form validation — and I’m steadily moving toward deeper topics like events, ES6 features, and asynchronous operations.

responsive design

Creating responsive layouts is one of my top priorities. I use media queries, relative units, and mobile-first design principles to ensure websites look and function well on all screen sizes — from desktops to smartphones. I focus on making sure the user experience remains consistent, accessible, and visually clean across devices.
